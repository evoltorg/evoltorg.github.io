Do you work on a Macintosh?
Do you use JavaScript in writing web pages?

Now you can use JavaScript to script behaviors on your Macintosh just like you can using AppleScript.

LateNight Software has released JavaScript OSA.

http://www.latenightsw.com/freeware/Java ScriptOSA/index.html

It's based on the Mozilla JavaScript 1.4.2 release (Rhino?) and will let you write scripts to automate your Mac.

Be careful though, as it's only at 1.0d4 (the "d" stands for "Danger Will Robinson!").

Have fun!
