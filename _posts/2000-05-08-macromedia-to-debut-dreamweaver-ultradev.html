Macromedia, Inc. announced today the release of Macromedia Dreamweaver UltraDev, a visual Web application development tool

which looks to compete with products such as Microsoft's Visual InterDev and Adobe's GoLive.

With support for Cold Fusion Markup Language, Active Server Pages, and Java Server Pages, UltraDev jumps out as the front runner in cross-language development tools. UltraDev also has functionality to link to any ODBC, ADO, or JDBC datasource and can take advantage of COM objects and JavaBeans.

Check out the press release from Macromedia for more marketing info. It sounds like this is built on top of Drumbeat, which Macromedia purchased some time ago and has all the functionality of Dreamweaver 3. Sadly, only Mac and Windows ports at this time. More information as it becomes available..
