\nIt's been almost three years in the making and surrounded by controversy the entire time. Thousands of people have helped build it, and thousands more have criticized it. The official 6.0 release of Netscape Navigator from AOL/Netscape has been released today.\n

\n

\nThis is the first official release of the browser that is built upon freely available, open source Mozilla technology. There are lots of new features in this release which you can read about, or you can download the browser directly and see for yourself. Once we start using and testing it, we will post more information about this long-awaited browser.\n
