--- layout: post permalink: node/10781 --- ColdFusion 5.0 has been released by Macromedia. Some of the new features include:\n
    \n
  1. User defined functions\n
  2. Charting (simple, but nice)\n
  3. CFDUMP, the coder's best friend. This used to be in Spectra only, but now it's in the server proper. Not one day goes by where I don't use this tag.\n
  4. Query of Queries\n
  5. Flushed output\n
  6. Server reporting enhancements\n
  7. Server archiving. This is a really cool feature where you can backup both code and server settings into a simple file that can be deployed to new machines. This makes working in a cluster much easier.\n
  8. Server probes. A cool way to check the health of your system and perform actions when things go to heck.\n

There is a lot more, so I suggest you check Allaire.com for more information.

In general, this is a cool release both on the code side, as well as the management side. In fact, I think it's a much bigger release for how it aids and supports the management of web apps.
