--- layout: post permalink: node/60610 --- What Is XHTML?

* XHTML stands for EXtensible HyperText Markup Language\n * XHTML is aimed to replace HTML\n * XHTML is almost identical to HTML 4.01\n * XHTML is a stricter and cleaner version of HTML\n * XHTML is HTML defined as an XML application

The Most Important Differences between XHTML And HTML

* XHTML elements must be properly nested\n * XHTML documents must be well-formed\n * Tag names must be in lowercase\n * All XHTML elements must be closed\n * Attribute names must be in lower case\n * Attribute values must be quoted\n * Attribute minimization is forbidden\n * The id attribute replaces the name attribute\n * The XHTML DTD defines mandatory elements

HTML 1.0 specifies three XML document types: Strict, Transitional, and Frameset.

XHTML Transitional DTD\n Use this DTD when you need to use XHTML's presentational features because your readers don't have browsers that support Cascading Style Sheets (CSS):\n

XHTML Frameset DTD\n Use this DTD when you want to use frames!\n
