The Apache Project has released the first public alpha of Apache 2. \n

Apache 2.0a1 "includes several new enhancements, including the new Apache Portable Run-time and the new Multi-processing modules," according to Apache.

You can read a list of New Features, or download the alpha to try it out.

The alpha is available as source code only, not as binaries - so you will need a compiler to play with it. This is hardly surprising, considering the alpha is described as a "developer-only alpha release with numerous bugs and new features. Guaranteed to give you the best roller coaster ride of the year."
