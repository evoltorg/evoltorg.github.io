The technology is based around using DirectX in conjunction with existing web t" --- The man responsible for creating Microsoft's DirectX multimedia API has developed a new technology called Wild Tangent. The technology is based around using DirectX in conjunction with existing web technologies such as CGI, Java and ActiveX, to enable the streaming of high quality multimedia and 3D content into a users web browser. Because Wild Tangent is different from other existing web and multimedia platforms in that it allows developers to stream the same content found in DirectX games and multimedia applications to be streamed over the web, thus expanding the possibilities for content creation immensely. Check out the Wild Tangent web site for more info.
