Position: Web Developer

Position Type: Full Time

Description

The Web Developer will be responsible for implementing web site and web application designs using XSLT and/or ASP.NET. The Web Developer will be responsible for maintaining internal code standards and W3C standards as part of this implementation and should be conversant in HTML, CSS, and JavaScript.

In this position, the candidate should expect to:

  • Integrate designs and applications into web-based interfaces. This may include a content management solution and/or enterprise applications.

  • Provide feedback to application design team to improve software quality.

  • Develop templates and standards to be utilized across projects.

  • Participate in corporate standards and development process.

  • Work as an on-going development resource on multiple simultaneous projects.

  • Conduct work at local client sites when necessary.

Requirements:

All candidates should meet the following requirements:

  • In-depth knowledge of HTML, XHTML, CSS, JavaScript, and ability to code to W3C standards.

  • A working knowledge of Adobe Photoshop and Macromedia Flash.

  • Practical experience with XSLT and basic knowledge of ASP.NET are preferred.

  • Ability to structure time flexibly across many projects.

  • Ability to learn from constructive criticism.

Application Process:

Candidate should submit résumé, citing at least three examples of sample work. Each sample should include a detailed description of the role the candidate had in creating the sample sites. Samples may include system designs, web sites or code snippets.
