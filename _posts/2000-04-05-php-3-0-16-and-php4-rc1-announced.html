Lots of news from the developers of the popular PHP Hypertext Preprocessor this week.

\n

First up, the first release candidate for PHP 4.0 has been announced, making the final 4.0 version one step closer. Available for Unix and Win32, this release also has support of Java add-ons, and lots of other nifty features. Head to the download page and give it a whirl.\n

\n

\nAlso this week, version 3.0.16 was released which is a self described bug fix release, although OpenBSD support has been 'improved considerably'. Check out the changes for yourself and then download away from your favorite mirror.\n
