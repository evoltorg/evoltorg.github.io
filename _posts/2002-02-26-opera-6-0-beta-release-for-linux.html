Opera Software \nannounced today the initial beta of Opera 6.0 for the Linux operating system. Version 6\nfollows the popular version 5 release of the Opera browser for Linux, which according to the press release announcing the beta, \nhas been downloaded over a million times to date.

New features include a Mozilla inspired cookie manager for viewing and editing cookies set by web sites,\nimproved skinning and themes, Netscape style plug in support for Flash, Acrobat, Real, Quicktime and others,\nand 'mouse gestures' which untill now were only available from the Windows version of the software.\n

\n

You can download the beta release from the Opera Software website,\nor view a screenshot of the new release (opens in new window).

\n

Opera can be downloaded and used for free, but advertising is attached. \nThe ad-free version of Opera costs $39, although those who purchased version 5 of Opera will\nbe able to upgrade for free to 6.0 beta. If you purchased version Opera version 4, there's a $15 charge\nto upgrade.\n
