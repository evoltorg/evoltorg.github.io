\nInstead of going through the usefulness of the breadcrumb navigation concept all over again, I'll just refer you to Martin's article, Breadcrumbs for All. It's a great tutorial for those of us with Perl and Unix/Apache servers, but it leaves out those of us using ASP on IIS. It also leaves out a few others, but I'm just doing ASP here, so cut me some slack.\n

\nYou can see a sample of this code in action at the Kenan Center site (the Recreation section has the most depth right now). This site has only a few levels of directories, but it is still just as useful on that site as on extremely deep sites.\n

\nOne thing to keep in mind is that you have to name directories in plain english (or your human-readable language of choice) since it uses the directory names for the breadcrumb. This function will capitalize the words in the directory names, and it replaces all underscores with spaces for display. It does not handle special characters, but then again, neither does the Windows file system (at least not extended characters). It's a simple matter of using a replace() function against a pre-defined list of characters to enable that, however.\n

\nThe code should be created as an include file called breadcrumb.asp and stuffed into your general includes directory. The page that calls it should have a variable called PageTitle that is the name of the page. This variable is used by breadcrumb.asp (and if you're smart, the file calling it) to list the current page. If the variable is blank, however, it just uses the words "Current Page." It's also worth noting that this script goes right to the root of the site with its links, which may or may not be what you want.\n

\nFunction BreadCrumb(FullPath)
Do Until instr(1,FullPath,"/") = 0
	'## Create an array of letters in the alphabet.\n	Letters = array("a","b","c","d","e","f","g","h","i","j","k","l","m","n","o","p","q","r","s","t","u","v","w","x","y","z")
	'## split on the /\n	tmpPath = mid(FullPath,1,instr(1,FullPath,"/")-1)\n	strTmpPath = Trim(tmpPath)\n	DirPath = DirPath & strTmpPath & "/"
	'## upshift the first character\n	firstLetter = ucase(mid(strTmpPath,1,1))\n	strTmpPath = firstLetter & mid(strTmpPath,2,len(strTmpPath))
	'## replace udnerscores with spaces and upshift the following character\n	for each letter in letters\n		strTmpPath = Replace(Trim(strTmpPath),"_" & lcase(letter)," " & UCase(letter))\n	next
	'## split the next one out	\n	FullPath = mid(FullPath,instr(1,FullPath,"/")+1,Len(FullPath)-Len(tmpPath))
	'## separate them with >> symbols\n	IF strTmpPath = "" THEN\n		response.write "<a href=""/"" style=""text-decoration:none"">Home Page</a>"\n	ELSEIF strTmpPath = "Home" THEN\n	ELSE\n		response.write " &gt; <a href=""" & DirPath & """ style=""text-decoration:none"">" & strTmpPath & "</a>"\n	END IF\nLoop
IF PageTitle = "" THEN\n	response.write " : Current Page"\nELSE\n	response.write " : " & PageTitle\nEND IF
End Function %>\n

\nTo call this code in your page, you need to call the include file and (if you want) set the variable PageTitle as something. For example:\n

\n<% PageTitle = "About Us" %>\n<!--#include virtual ="/includes/breadcrumb.asp"-->\n<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN"\n                "http://www.w3.org/TR/REC-html40/loose.dtd">
<html lang="en-us">\n<head>
<title><% = PageTitle %></title>\n[...]\n

\nWhen you are ready to call the function on the page to display the breadcrumb trail, simply call it like this:\n

\n<% = BreadCrumb(Request.ServerVariables("PATH_INFO")) %>\n

\nThe code can easily be tweaked, and some things changed or modified (like the Letters array, or setting the PATH_INFO server variable as a variable early in the script) depending on your needs. You can modify the HTML that displays the breadcrumb in the include file so it matches your site styles as well.\n
