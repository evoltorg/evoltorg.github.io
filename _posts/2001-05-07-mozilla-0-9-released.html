\nMozilla.org released milestone 0.9 yesterday, inching ever closer to its goal of a feature complete, standards compliant, open source web browser.\n

\n

\nThe 0.9 milestone release is mostly a performance release with some significant rewrites in a couple of spots. Some of the biggest changes you'll notice:\n

  • \nAutomatic Proxy Configuration has been implemented.\n
    • \n
  • \nPersonal Security Manager 2.0 with improved performance and new UI.\n
    • \n
  • \nMailNews front end has been overhauled with a huge performance improvement.\n
    • \n
  • \nBrowser and Mail now utilize a new cache, and new view manager for improved performance and correctness.\n
    • \n
  • \nLate loading of java has been implemented for improved startup performance and footprint.\n
    • \n
  • \nLong-click means of invoking contextual menus on Mac now implemented.\n
    • \n
  • \nImage rendering library was rewritten from scratch for increased performance.\n
    • \n
\n

\n

\nThe final beta milestone is expected to hit by the end of May. Any release after 0.9.1 could be the official 1.0 release of Mozilla if things go well. So download a copy and use the talkback feature to report the remaining bugs! You can download either from mozilla.org, or find a copy for your platform in the browser archive.\n
