Allaire Corp has released its Forums product to the developer community under the 'Open Source' banner this week.

You can find the source code (Cold Fusion templates) at the forumspot.org website. Basically, they changed the license terms for Forums to give the end user more freedom in how they use that, but its far from 'true' open source software.

Even Slashdot picked up the news, and there are some good discussions there. Sadly, there are only Windows NT and Solaris ports for Forums currently available. You'd think that Allaire would have a port for its one 'Open Source' product out for the only true Open Source OS it supports, Linux, but it hasn't happened yet. The forumspot.org website claims it will have ports for both Linux and HP-UX out sometime this spring.
