--- layout: post permalink: node/768 --- Here is a very simple cgi script to send the form \noutput to the target mail boxes:\n

One - cut and paste the code below into "notepad"\n

Two - edit the following lines:\n

\n  /full/path/to/cgi-bin/\n

Change to the "full path" to your cgi dir.\n

\n  $varSendFormDataTo = "user1\@Foo.com";\n

\nChange Foo.com to your domain and user1 to the target user. \nMake sure you have \@ and not a 'bare' @.\n

\n  $varSentFrom = "WebServer\@Foo.com";\n

\nIf you want to send to more than one user, add a comma and \nthe new name@foo.com.\n

\n  $varSubject = "Web Form Comments";\n

\nChange the Subject variable to what you want displayed on the email.\n

\n  $varJumpToWebPage = "http://www.foo.com/index.html";\n

\nThis is the page you are redirected to - can be \nany valid page anywhere on the internet.\n

\nVerify that sendmail is in your\n/usr/sbin/\ndirectory. The quickest way is to type\n'whereis sendmail'\n

\nThree - Save your file in your cgi directory. \n

\n  chmod 755 <file name>.\n

\nFour - Get cgi-lib.pl from \n\nhttp://cgi-lib.stanford.edu/cgi-lib/\n

\nSave in your cgi-bin directory.\n

Five - Number your form elements. e.g. add \na #- to the front of each name (FullName \nbecomes 1-FullName, Email become 2-Email).\nThe program expect numbers and splits on \nthe first '-'. This allows you to number \nthe form elements in the order you want them \nto appear in the email.\n

Six - Don't forget to set your action \nto "/cgi-bin/qd-form.cgi" and remove any \nform encoding types you have set.\n

Good Luck.\n

\n\n

\n#!/usr/bin/perl\n#\n# Quick & Dirty Form Processing....\nBEGIN {unshift @INC, '/full/path/to/cgi-bin/' };\n require "cgi-lib.pl";
##\n## User Set Variables\n$varSendFormDataTo = "user1\@Foo.com";\n$varSentFrom = "WebServer\@Foo.com";\n$varSubject = "Web Form Comments";\n$varJumpToWebPage = "http://www.foo.com";
##\n## System Set Variables\n$varSendMail = "/usr/sbin/sendmail -t -n";\n$today = `date`;\n$returnvalue=&ReadParse(\%input);
open (MAIL,  "| $varSendMail") || die "cannot open mail program";\n    print MAIL "To: $varSendFormDataTo \n";\n    print MAIL "From: $varSentFrom \n";\n    print MAIL "Subject: $varSubject
";\n    print MAIL "$today
";
    foreach $var(sort keys %input) {\n        if (lc($var) eq "submit") {next;}\n        $varTrimmed = splut(/-/,$var,2);\n        print MAIL "$varTrimmed -- $input{$var}\n";\n    }
print MAIL " 
";\nclose(MAIL) || die "mail pipe exited $?";
print "Location: $varJumpToWebPage
";\n             \nexit;
