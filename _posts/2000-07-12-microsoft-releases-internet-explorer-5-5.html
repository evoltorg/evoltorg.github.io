Microsoft has released Internet Explorer 5.5. This latest version sports virtually no UI changes aside from a print-preview feature. Its support for W3C standards such as HTML4 and CSS1 has improved, but is still incomplete.

IE 5.5 has some other features of interest to web developers, such as built-in support for W3C's SMIL Boston multimedia standard, element behaviors, and <frame> and <iframe> elements that support transparency and z-indexing with other content.

For further information on IE 5.5's new features, see http://www.microsoft.com/presspass/features/2000/jul00/07-12ie5.5.asp. To download the IE 5.5 setup utility, see http://www.microsoft.com/windows/ie/default.htm.
