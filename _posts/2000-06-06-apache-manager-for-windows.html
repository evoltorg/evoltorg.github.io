For those that run the Apache Webserver on the Windows platform, it can be a pain to start, stop, and restart Apache when setting it up or testing different configurations.

To make things easier, Apache developer Brian Moon wrote a simple application that sits in the Windows task tray and allows you to start, stop, and restart Apache easily. You can find more out about the application at his Apache Manager for Windows Web page.
