Another in the line of the 4.7x family of browsers from Netscape has been released. New features include Flash 4.0 support for Linux and Solaris, RealPlayer 7 for folks in the Windows world, and a 2.6 release of the Winamp MP3 player. You can get more info on this upgrade by checking out the release notes.

This upgrade also apparently fixes a security hole that allows JPEG comments to cause a buffer overflow, allowing arbitrary code to be executed as the user thats using Netscape. Check out Red Hat's errata page for RPM downloads, or get the full scoop at securityfocus.com

Downloads can be had from the Netscape FTP site or from our even-expanding browser archive
