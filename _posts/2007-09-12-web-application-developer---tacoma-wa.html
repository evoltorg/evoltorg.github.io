POSITION SUMMARY

SiteCrafting is a website design and web application development company that specializes in custom web-based solutions for mid- to large-size businesses, counties, and municipalities. The company has been headquartered in the Tacoma, Washington area since 1995.

We are currently seeking a Web Application Developer to join our talented web development and design team. This position is responsible for developing, maintaining and supporting existing SiteCrafting customers' websites.

REQUIRED EXPERIENCE

Two-year Associate’s or four-year Bachelor’s Degree (B.A./B.S.) in Computer Science or equivalent combination of education and experience.

Software development experience in a web environment with a strong knowledge of:

  • PHP and/or .NET

  • MySQL and/or MS SQL

  • CSS

  • JavaScript

  • CVS development environment

  • Windows, Macintosh and Unix server environments

REQUIRED SKILLS

Demonstrated ability to do the following:

  • Effectively manage workload on multiple projects and meet all required timelines

  • Work independently, but able to effectively function as a member of a team

  • Work cooperatively with teammates, vendors and clients

POSITION LOGISTICS

  • Location: Tacoma, WA

  • Full-time position, Mon. - Fri., 8-5pm

  • Salary: DOE

  • Reports to Senior Web Developer

  • No contractors

  • No telecommuting
