\nNow that the Webreference \n Hierarchical menus have gone commercial, it is time for some of us to consider \n good alternatives for developing those DHTML drop-down menus.\n

\nIn this quick \n posting (hesitate to dignify this as an "article"), I will share some \n options from my stash, with some relevant preliminary comments. This is not supposed to \n be TheGreatComparisonOfMenuScripts or a comprehensive rehash of search results from \n Google... it is just a listing of possible options that I have accumulated on \n my local drive. Hope this is useful.\n

\nBefore we begin, I have tested these menu scripts with the following browsers \n using a Windows XP build 2600 and a Windows 2000 box —\n

    \n
  • IE 4x - 6x
    • \n
  • Netscape 4.77
    • \n
  • Netscape 6.2
    • \n
  • Mozilla 0.9.8 (Mozilla/5.0; Gecko/20020204)
    • \n
  • Opera 6.01 Build 1014
    • \n
