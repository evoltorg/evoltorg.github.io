--- layout: post permalink: node/200 --- Centraal has signed a deal with Microsoft in order to get its RealNames service integrated with the Microsoft Network and as a default search in Internet Explorer. As well, a deal with AOL is in the works which may also see RealNames incorporated into AOL's Communicator web browser.

\nRealNames is a database of plain-language terms that map to web sites. Web users will no longer have to remember long URLs or obscure sequences of mouse clicks in order to get the information that they are looking for.
